# cybersecurity-audit-gift
 Beginner Cybersecurity Audit Project

 Name

Nwafor Gift

 Project Title

Personal Laptop Cybersecurity Audit

 Objective

The aim of this project is to identify common cybersecurity weaknesses on a personal laptop and recommend practical solutions to improve its security.

 Scope

This audit focuses on my personal Windows laptop.

 Tools Used

- Windows Security
- Windows Settings
- Microsoft Defender
- Windows Firewall
- Windows Update
- Google Chrome Password Manager

 Files in this Repository

- README.md
- checklist.md
- findings-report.md
- recommendations.md
- screenshots/

 Key Findings

- Password protection is enabled.
- Microsoft Defender Antivirus is active.
- Windows Firewall is enabled.
- Some Windows updates are pending.
- Screen lock is enabled.
- No suspicious applications were detected.

 Conclusion

This project improved my understanding of basic cybersecurity practices and demonstrated how routine security checks can help protect a personal computer from common cyber threats
