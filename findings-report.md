Findings Report

Target Audited

Personal Laptop

 Summary

I carried out a basic cybersecurity audit on my personal laptop to evaluate its security posture and identify common security weaknesses.

 Findings

1. A password is enabled for user authentication.
2. Microsoft Defender Antivirus is installed and active.
3. Windows Firewall is enabled.
4. Windows Update shows that some updates are pending.
5. No suspicious applications were found during the inspection.
6. Screen lock is enabled to prevent unauthorized access.
7. Browser passwords are stored securely.

 Risk Level = Medium Risk

 Why?

The laptop has several important security features enabled, including password protection, antivirus, and firewall. However, pending system updates could expose the device to known vulnerabilities until they are installed.
